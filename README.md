# Weather App
A basic weather web application built with vanilla Javascript and webpack.\
Created as part of The Odin Project full stack developer curriculum\
No AI was used in its production.

## Deployed Link
https://ravenovus.github.io/Weather-App/

## Features
Fetches weather information about a target city\
using the Visual Crossing API and presents today's\
detailed information, and more limited information for\
6 more days.\
Includes icons for every weather presented by the API\
and 5 generic weather images for the background.

## Technology Used
- Javascript (ES6+ Modules)
- Webpack Bundler
- CSS
- Visual Crossing API
